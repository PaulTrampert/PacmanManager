Implement the fixes or features described in GitHub issue [#$ARGUMENTS](https://github.com/PaulTrampert/PrivatePacmanRepo/issues/$ARGUMENTS).

The workflow for implementing a solution to an issue is as follows:
* Understand the requirements of the issue
* Research the existing codebase
* Plan test cases that will validate a solution meets the criteria of the issue
* Plan a solution that will pass the test cases
* Create a git branch from the head of `main` to implement the changes in
* Implement the plan
* Commit and push the changes
* Create a Pull Request.

When performing the above, ensure your work adheres to CONTRIBUTING.md. Use AGENTS.md to understand the tools available to you, in addition to your system prompt.
